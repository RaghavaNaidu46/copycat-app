# Copycat – Support & Legal

This repository hosts public documents for **Copycat**, a macOS clipboard history manager available on the Mac App Store.

## For users

- **[Privacy Policy](PRIVACY_POLICY.md)** – What data the app stores and how it is used. All data stays on your Mac; nothing is sent to the internet.

## Support

For questions, feedback, or support:

- Open an [Issue](https://github.com/raghav-6220/copycat-app/issues) on this repo, or  
- Use the **Support** link in the app (Copycat → Preferences → Advanced → Support).
